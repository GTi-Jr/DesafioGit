O QUE É GIT

Git é um sistema de controle de versão de arquivos. Nele, é possível gerenciar projetos nos quais diversas pessoas podem contribuir ao mesmo tempo, modificando e desenvolvendo arquivos, armazenando-os em um servidor e sem o risco de perder tais modificações.

O QUE É GITHUB

Github é uma plataforma de armazenagem em nuvem de projetos para controle de versão usando o git.
