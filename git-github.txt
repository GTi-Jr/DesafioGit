O que é GIT?
É um sistema de controle de versões, que serve para gerenciar todas as alterações realizadas no arquivo ao longo do tempo, sem haver sobrescrição.
O que é GITHUB?
É um site que serve para hospedar projetos que possui algumas características de redes sociais que permite seguir outros usuários e acompanhar seus projetos. O github utiliza o git para controle de versão.
=======
o git é uma forma de criar e desenvolver projetos de forma versificada e guardá-los na nuvem, de forma que outras pessoas possam usar os códigos e modificá-los.
O que é github:
Github é a plataforma que armazena códigos enviados pelo git e permite que terceiros os visualizem.
